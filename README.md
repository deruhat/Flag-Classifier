# Flag Recognizer

Keras multi-class classifier for 17 country flags:
- Afghanistan
- Albania
- Algeria
- Andorra
- Angola
- Antigua & Deps
- Argentina
- Armenia
- Australia
- Austria
- Azerbaijan
- Bahamas
- Bahrain
- Bangladesh
- Barbados
- Belarus
- Belgium

#### Tensorboard graphs are in the 'logs' directory for visualizing training progress.
